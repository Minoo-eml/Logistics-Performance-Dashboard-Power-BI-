# Logistics Performance Dashboard

## Overview
This project analyzes logistics operations across hubs, drivers, and vehicles using Power BI.

## Key KPIs
- On-time delivery rate: 79.85%
- Customer satisfaction: 87.56%
- Average delivery time: 35.5

## Key Insights
- Some hubs are overutilized while others are underutilized
- Delivery delays are linked to specific drivers and vehicle types
- Vehicle condition impacts delivery performance

## Tools
- Power BI
- DAX
- Power Query

## Business Impact
This dashboard helps identify bottlenecks and supports better decision-making in logistics operations.
